# Burton Makes

Public engineering project archive for hardware, wearables, 3D printing, bioengineering, and AI systems.

Live site:

https://burtonmakes.github.io

## Files

- `index.html` — main website page
- `style.css` — site styling

## How to edit

Edit `index.html` to update text, project cards, and sections.

Edit `style.css` to change colors, spacing, typography, and layout.

## How to add a project card

Copy one of the existing `<article class="card">...</article>` blocks in `index.html` and update:

- status
- project title
- project description
